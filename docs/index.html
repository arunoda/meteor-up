Why Meteor UP?

Ship your application with less headache and overhead.
Deploy from anywhere. Meteor Up runs on Windows, Mac, and Unix.
You are free to deploy to any Ubuntu server you can ssh into. You are not tied to a specific platform or infrastructure as a service.

Easy Deplyment Using Docker

    Docker Allows For Consistent Results

Using docker we know that we are using the same meteor enviornment, same proxy, same everything every time. Once you setup your mup config, you no longer have to worry about will this work on this other server. Docker takes care of that for us.



    Custom Configurations

Because we can mount volumes in Docker, that means we can pass configuration files to the underlying containers. such as custom Nginx configurations, meteor settings, and more



    SSL Certificates

Using the same volume mapping for configurations we can send SSL Certificates to a Nginx docker container. This allows us to reverse proxy the application and serve it's content over https. The outside world does not directly talk to your application, but to the proxy. This is a more secure way to deploy your app.



Heads up! Now with Let's Encrypt integration, you don't even need to purchase an SSL. Meteor Up can auto generate that for you. For more information please look at the github documentation and the let's encrpyt website.
